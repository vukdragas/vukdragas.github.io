Demo of User Menagement System:

Full Video Description 
User Management: 
• Create new users with name, age, ID, and group assignments 
• Modify existing user information (name and age) 
• Delete users from the system 
• View individual user details or all users in the system  
Group Management: 
• Predefined groups with specific IDs (root, system users, operators, observers) 
• Each user can belong to up to 4 groups (MAX_GROUPS) 
• Ability to add users to additional groups  
System Operations: 
• Count total number of users in the system 
• Export all user data to a text file 
• Display system information in a tabular format  
Initialization: 
• System starts with a default "root" user with admin privileges 
• Root user is automatically assigned to all predefined groups  
Technical Implementation: 
• Uses linked list data structure for storing user records 
• Implements basic memory management with malloc/free 
• Provides console-based menu interface 
• Supports file I/O for data export  
This project demonstrates fundamental C programming concepts including structures, 
pointers, dynamic memory allocation, file handling, and linked list operations, 
making it an excellent example of a console-based management system.
